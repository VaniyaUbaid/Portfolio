# 🌐 Personal Portfolio Website

This is my personal portfolio website built using **HTML, CSS, JavaScript, and Bootstrap**. It showcases my projects, skills, and provides a way to connect with me.

---

##  Features

* ✨ Responsive design (works on all devices)
* 🌙 Dark mode toggle
* 🔽 Smooth scrolling navigation
* 🧩 Projects section with images
* 📬 Contact form integration

---

##  Tech Stack

* HTML5
* CSS3
* JavaScript
* Bootstrap

---

##  Project Structure

```

```
